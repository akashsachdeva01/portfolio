# Akash Sachdeva — Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 🌐 Live Site

Visit: [akashsachdeva01.github.io/portfolio](https://akashsachdeva01.github.io/portfolio) *(once deployed via GitHub Pages)*

## ✨ Features

- **Premium dark theme** with glassmorphism cards and animated gradient orbs
- **Typed text animation** cycling through professional roles
- **Animated counters** for career stats
- **Scroll-triggered animations** (AOS-style)
- **Mouse parallax** on hero background elements
- **Fully responsive** — works on mobile, tablet, and desktop
- **SEO optimised** with meta tags and semantic HTML

## 📄 Sections

1. Hero — Name, typed roles, career stats
2. About — Key strengths and summary
3. Experience — Full work timeline (Ericsson, Nokia, TCS/Cisco, Wipro)
4. Skills — Protocols, automation, tools, vendors
5. Education — BITS Pilani MS, Bangalore University BCA
6. Contact — Email, LinkedIn, Phone

## 🚀 Running Locally

Simply open `index.html` in any modern browser — no build step required.

## 📦 Deploying via GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
